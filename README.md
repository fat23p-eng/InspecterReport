# InspecterReport
รายงานผลการเข้าตรวจการสหกรณ์ โดยทีมตรวจสอบระดับจังหวัด
ระบบรายงานผลการตรวจการสหกรณ์พร้อมใช้งานแล้วครับ มีฟีเจอร์ครบตามที่ต้องการดังนี้
📊 Dashboard
สรุปสถิติสำคัญ: จำนวนสมาชิก, กำไรสุทธิ, ความคืบหน้า
Progress bar รายการตรวจสอบทั้ง 16 ข้อ
สถานะทีมผู้ตรวจการแต่ละคน
📝 กรอกแบบรายงาน
แบ่งเป็น 4 Tab: ข้อมูลส่วนหัว / รายการ 1-8 / รายการ 9-16 / ผู้ลงนาม
รายการตรวจสอบครบทั้ง 16 ข้อตามไฟล์ Demo พร้อมข้อมูลจริง
กรอกร่วมกันหลายคน — แต่ละคนกรอกเฉพาะส่วนที่รับผิดชอบ
บันทึกอัตโนมัติในเครื่อง
📋 ข้อมูลสัญญาเงินกู้
สัญญาเงินกู้ฉุกเฉิน 21 สัญญา (เอกสารแนบ 1)
สัญญาเงินกู้สามัญ 10 สัญญา (เอกสารแนบ 2)
📥 นำเข้าข้อมูล / 📎 เอกสารแนบ
อัปโหลดไฟล์แนบ drag & drop
โหลดข้อมูล Demo ได้ทันที
👁️ Preview & ส่งออก
Preview ก่อนส่งออก เห็นรูปแบบรายงานจริง
ส่งออก Word (.doc) ที่มีตารางและลายมือชื่อครบ
พิมพ์ ได้โดยตรงจากระบบ
💳 เมนูข้อมูลเงินกู้ — แยกประเภทพร้อมแก้ไขได้
แบ่งเป็น 4 Tab: 🔴 เงินกู้ฉุกเฉิน / 🟠 เงินกู้ฉุกเฉิน ATM / 🔵 เงินกู้สามัญ / 🟢 เงินกู้พิเศษ
ทุกช่องในตาราง แก้ไขได้โดยตรง (คลิกแล้วพิมพ์ได้เลย)
ปุ่ม + เพิ่มสัญญา และ ลบ รายการได้
ปุ่ม บันทึก บันทึกลง localStorage
📂 เมนูเอกสารราชการ 3 รายการใหม่
📋 คำสั่งแต่งตั้งผู้ตรวจการ — กรอกเลขที่คำสั่ง วันที่ ผู้ออกคำสั่ง พร้อม drag & drop ไฟล์
📨 หนังสือแจ้งสหกรณ์ — กรอกเลขที่หนังสือ วันที่ สหกรณ์ที่รับ กำหนดวันเข้าตรวจ พร้อมอัปโหลดไฟล์
🗂️ เอกสารอื่นๆ — เพิ่มรายการเอกสารพร้อมประเภท หมายเหตุ และแนบไฟล์แต่ละรายการ
